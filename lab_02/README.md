## Опис завдання
Проєкт складається з двох частин:
**Частина 1:** Завантаження сирих даних VHI-індексу з NOAA для адміністративних областей України, їх очищення (data cleaning), зміна індексації на українську абетку та реалізація функцій для формування вибірок (за роками, областями, пошук екстремумів).
**Частина 2:** Профілювання та аналіз датасету `Individual Household Electric Power Consumption`. Виконано фільтрацію за умовами, нормування, стандартизацію, обчислення кореляції (Пірсона/Спірмена) та One Hot Encoding. Час виконання процедур виміряно за допомогою модуля `timeit`.

## Вимоги до системи
- **ОС:** Windows, macOS або Linux
- **Інтерпретатор:** Python 3.10 або вище 
- **Пам'ять:** Мінімум 4 ГБ ОЗП (для завантаження датасету другої частини)
- Редактор з підтримкою .ipunb

## Інструкція із запуску та встановлення залежностей

Створіть та перейдіть у директорію проєкту:  
  cd lab-2
Створіть віртуальне середовище:
python -m venv venv

Активуйте віртуальне середовище:
Для Windows:
venv\Scripts\activate

Для macOS/Linux:
source venv/bin/activate

Встановіть необхідні залежності:
pip install -r requirements.txt

Підготовка даних для Частини 2:
Завантажте датасет household_power_consumption.txt.

Розмістіть його у папці data/ всередині директорії проєкту (папка буде ігноруватись Git).

Запустіть Jupyter Notebook:
jupyter notebook

Після запуску відкрийте відповідні файли .ipynb для перегляду та виконання коду.
